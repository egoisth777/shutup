# Final Arca decision

Shutup is a clean fork of Caveman. It keeps only terse-response activation, four static skills, and installation for Claude, Codex, Google Antigravity, Pi, and Oh My Pi. Legacy statistics, statusline, MCP shrink, compression, crew, OpenCode, OpenClaw, duplicated plugin trees, and generated distribution artifacts are retired. Existing Caveman installations are not migrated or modified.

Arca is retired at this fork boundary. Its final governance decision is preserved here because the new repository favors code, tests, and short maintainer docs over a mirrored knowledge tree. Future architecture and behavior changes belong in CLAUDE.md and user-facing docs; tests enforce runtime contracts.
